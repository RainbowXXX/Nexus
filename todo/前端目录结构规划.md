# Tauri + Vue + TypeScript 聊天软件客户端前端目录结构规划

## 项目根目录结构

```
reNexus/
├── src-tauri/          # Tauri 后端代码
├── src/                # Vue + TypeScript 前端代码
├── public/             # 静态资源
├── todo/               # 项目规划文档
├── package.json        # 项目依赖
├── tsconfig.json       # TypeScript 配置
└── README.md           # 项目说明
```

## 主要界面功能规划

### 1. 登录界面 (Login)

- 用户名/邮箱输入
- 密码输入
- 记住密码选项
- 登录按钮
- 注册链接
- 忘记密码链接

### 2. 主界面 (Main)

- 聊天列表区域
  - 搜索框
  - 聊天列表项
  - 新建聊天按钮
- 聊天窗口区域
  - 聊天对象信息栏
  - 消息显示区域
  - 消息输入框
  - 表情按钮
  - 文件发送按钮
- 侧边工具栏
  - 用户头像
  - 设置按钮
  - 通知按钮

### 3. 设置界面 (Settings)

- 账户设置
  - 个人资料修改
  - 密码修改
  - 头像上传
- 通知设置
  - 消息通知开关
  - 声音设置
- 外观设置
  - 主题切换（亮色/暗色）
  - 字体大小调整
- 通用设置
  - 语言切换
  - 数据清理
  - 关于页面

## 技术栈建议

- 前端框架：Vue 3 + TypeScript
- UI 组件库：Element Plus
- 状态管理：Pinia (Vue 官方推荐的新一代状态管理工具)
- 路由：Vue Router 4
- 样式预处理：Sass/SCSS
- 桌面应用：Tauri
- HTTP 请求：Axios
- 图标：Element Plus Icons / @iconify/vue (图标库聚合器)
- 开发工具：Vite (构建工具)
- 代码规范：ESLint + Prettier
- Git 提交规范：Commitizen + husky
